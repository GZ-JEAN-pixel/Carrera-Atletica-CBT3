# 🏃 CBT3 - Carrera Atlética 2026

Sitio web oficial de la Carrera Atlética del Centro de Bachillerato Tecnológico No. 3 Otzolotepec.

## 📁 Archivos

```
├── index.html     ← Página principal
├── estilos.css    ← Todos los estilos del sitio
└── scripts.js     ← JavaScript (animaciones, navegación)
```

## 🚀 Ver en GitHub Pages

1. Ve a **Settings** → **Pages**
2. En *Branch* selecciona `main` y carpeta `/ (root)`
3. Guarda y espera ~1 minuto
4. Tu sitio estará en: `https://TU_USUARIO.github.io/NOMBRE_REPO`

## 💻 Ver en local

Abre `index.html` directo en tu navegador, o con servidor local:

```bash
python3 -m http.server 8080
```

Luego entra a: http://localhost:8080
